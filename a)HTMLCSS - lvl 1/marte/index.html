Já é possível fazer suas férias em Marte. A viagem é longa, mas a experiência é única!

Mas muita atenção: são necessários alguns itens preparatóriosantes de seguir a viagem

O que necessito para viajar?

Para viajar para Marte vai ser necessário:

  • Exames médicos gerais
  • Passaporte espacial
  • Assinar um termo de responsabilidade

Procedimentos antes da viagem:

  1. Escolher a data da viagem
  2. Efetuar o pagamento de 50% do valor da viagem
  3. Participar de duas semanas de treino gravitacional especializado
  4. Assinar o termo de responsabilidade

MUITO IMPORTANTE

A viagem só está disponível para maiores de idade
